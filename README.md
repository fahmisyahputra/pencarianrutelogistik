Cara Run app.py dan Install dependecy/modul non global dengan venv


Bikin venv

```
python -m venv venv
```

Aktifkan venv

```
# Cari file activate di venv, secara umum kyk gini :
./venv/Scripts/activate
```

Install modul

```
pip install -r requirements.txt
```

Run app.py
```
python app.py
```
